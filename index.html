Descripcion

El arroz chino es un plato versátil y sabroso que se puede preparar de muchas formas. A continuación, se presenta una receta fácil y rápida para preparar arroz chino con pollo.

Disfruta de un sabroso arroz frito con pollo para saciar tus antojos de comida china. Es un plato muy fácil de hacer y que estará listo en pocos minutos. Así, es la receta perfecta para una cena rápida entre semana.
